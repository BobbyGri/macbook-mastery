# 📘 Overview

## What is this repository?

**MacBook Mastery** is a structured and modular knowledge base designed to help developers and power users fully leverage macOS.

It combines:

* Keyboard layout knowledge
* System and application shortcuts
* Practical workflows
* Automation scripts
* Setup and configuration guides

The goal is not only to collect information, but to organize it in a way that reflects how real systems are designed: modular, scalable, and efficient.

---

## 🧠 Core Idea

Instead of treating macOS usage as a set of isolated tricks, this repository approaches it as a **system** composed of layers:

* **Input Layer** → Keyboards (how you interact)
* **Action Layer** → Shortcuts (how you act)
* **Process Layer** → Workflows (how you operate)
* **Execution Layer** → Scripts (how you automate)
* **Configuration Layer** → Setup (how you initialize)
* **Knowledge Layer** → Docs (how you understand)

---

## 🧩 Structure Overview

```bash
keyboards/   # Input (keyboard layouts and mappings)
shortcuts/   # Actions (shortcuts and commands)
workflows/   # Processes (step-by-step usage)
scripts/     # Execution (automation and tools)
setup/       # Configuration (system and dev setup)
docs/        # Knowledge (concepts and philosophy)
```

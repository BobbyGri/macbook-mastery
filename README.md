# 💻 MacBook Mastery

A structured and modular toolkit to master macOS productivity.

This repository centralizes keyboard layouts, shortcuts, workflows, and automation scripts designed for developers and power users who want to work faster, cleaner, and more efficiently on MacBook.

---

## 🎯 Purpose

The goal of this project is to:

* Improve productivity on macOS
* Standardize workflows across different keyboard layouts
* Provide reusable automation scripts
* Document efficient development and system usage practices
* Reduce friction when switching between languages and environments

---

## 🧩 Repository Structure

```bash
macbook-mastery/
├── keyboards/     # Keyboard layouts (US, DE, FR, MX, RU)
├── shortcuts/     # System and application shortcuts
├── workflows/     # Step-by-step productivity workflows
├── scripts/       # Bash / Zsh automation scripts
├── setup/         # macOS and development setup guides
└── docs/          # Concepts, philosophy, and references
```

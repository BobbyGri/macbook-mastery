# ⌨️ Keyboard Layouts 

## 🎯 Purpose

This section covers different keyboard layouts and how to use them effectively on macOS.

It also explains how to configure and switch between layouts.

---

## ⚙️ Keyboard Configuration

### Add or Change Keyboard Layout

**How:**

1. Open **System Settings**
2. Go to **Keyboard**
3. Click **Input Sources**
4. Click **Add (+)**
5. Select your desired layout (e.g., US, German, Spanish, French)

---

### Switch Between Layouts

Once multiple layouts are added:

* Use:

```text id="slp1ra"
⌃ + Space
```

Or enable the input menu in the menu bar.

## 🧠 Recommended Strategy

If you work with multiple languages:

* Use **US layout for coding**
* Use your native layout for writing

**Why:**

* Most programming symbols are easier in US layout
* Most documentation assumes US layout
* Reduces friction in development environments

---

## ⚡ Tips

* Keep only the layouts you actually use
* Learn the position of key symbols in each layout
* Avoid switching layouts unnecessarily

---

## 🔗 Layouts Available

* `US/` → English (United States)
* `DE/` → German
* `FR/` → French
* `LA/` → Spanish (Latin America)
* `RU/` → Russian

---

## 🔄 Cross-Layout Mappings

Sometimes the goal is not only to learn a keyboard layout, but to **interpret one layout from another**.

Example:

* You see or think in **US**
* You want to type in **Russian**

In that case, use the mapping guides in [Cross-Layout Mappings](_mappings/)

These documents provide a **1:1 base-key correspondence** between layouts.

They are intentionally limited to the main key positions and do **not** include advanced modifier combinations such as Option or Shift + Option.

For extended characters and advanced combinations, see the documentation of the destination layout.


---

## 🧠 Final Thought

Your keyboard layout is part of your system.

Choosing the right layout — and using it consistently — has a direct impact on your productivity.

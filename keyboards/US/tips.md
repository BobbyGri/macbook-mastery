# 💡 US Keyboard Tips 

## 🎯 Purpose

Practical tips to improve efficiency when using the US keyboard on macOS.

---

## ⚡ Use Spotlight as a Launcher

Instead of navigating manually:

```text
⌘ + Space → type app/file name → Enter
```

**Why:**
Faster and more efficient than browsing folders.

---

## 📂 Open Terminal in Current Folder

### Method 1: Drag & Drop

```bash
cd [drag folder here]
```

### Method 2: Right Click (Automator Service)

Create a workflow to open Terminal in the selected folder.

**Why:**
Reduces navigation steps and context switching.

---

## ✍️ Faster Navigation in Terminal

| Action        | Shortcut   |
| ------------- | ---------- |
| Move by word  | ⌥ + ← / →  |
| Start of line | Ctrl + A   |
| End of line   | Ctrl + E   |
| Delete word   | ⌥ + Delete |

---

## 🔁 Repeat Commands Efficiently

Use command history:

```bash
↑ / ↓ arrows
```

Search history:

```bash
Ctrl + R
```

**Why:**
Avoid retyping commands and speed up workflows.

---

## 🧠 Think in Workflows

Instead of isolated actions:

* Combine shortcuts
* Automate repetitive steps
* Reduce context switching

👉 Example:
Finder → Terminal → Script → Done

---

## ⚡ Minimize Mouse Usage

Prefer keyboard over mouse whenever possible.

**Why:**

* Faster interaction
* Less context switching
* More consistency across tools

---

## 🔄 Switch Keyboard Layouts Wisely

If you use multiple layouts:

* Use **US layout for coding**
* Use native layout for writing

**Why:**
Most development tools assume US layout.

---

## ⚠️ Common Mistakes

* Overusing the mouse instead of shortcuts
* Not customizing the environment
* Trying to memorize everything at once

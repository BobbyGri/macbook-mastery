# ⚡ US Keyboard Shortcuts 

## 🎯 Purpose

This document provides essential keyboard shortcuts to improve **speed, efficiency, and workflow** on macOS using the US keyboard layout.

Shortcuts are grouped by context to make them easier to learn and apply.

---

## 🧭 System Navigation

| Action              | Shortcut  |
| ------------------- | --------- |
| Spotlight Search    | ⌘ + Space |
| Switch Applications | ⌘ + Tab   |
| Mission Control     | Ctrl + ↑  |
| Show Desktop        | F11       |

---

## 📸 Screenshots

| Action                  | Shortcut          |
| ----------------------- | ----------------- |
| Capture entire screen   | ⇧ + ⌘ + 3         |
| Capture selected area   | ⇧ + ⌘ + 4         |
| Capture window          | ⇧ + ⌘ + 4 → Space |
| Open screenshot toolbar | ⇧ + ⌘ + 5         |

### Tips

* Screenshots are saved to the Desktop by default
* Use the toolbar (⇧ + ⌘ + 5) for screen recording
* You can select specific areas or windows

---

## 😄 Emojis & Symbols

| Action                     | Shortcut      |
| -------------------------- | ------------- |
| Open emoji & symbol picker | ⌃ + ⌘ + Space |

### Tips

* Works in most applications (Notes, Slack, browsers)
* You can search emojis by name
* Includes symbols and special characters

---

## 🧠 Pro Tips

* Use Spotlight (⌘ + Space) instead of navigating manually
* Combine shortcuts to create workflows
* Learn shortcuts incrementally — focus on high-frequency actions
* Reduce mouse usage to increase speed

---

## ⚠️ Common Mistakes

* Trying to memorize everything at once
* Overusing the mouse instead of shortcuts
* Not adapting shortcuts to daily workflows

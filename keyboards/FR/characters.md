# 🔤 FR Character Reference 

## 🎯 Purpose

This document provides a complete set of **copyable characters** for the French (AZERTY) keyboard layout.

Use this when:

* You cannot find a character on the keyboard
* You do not remember the key combination
* You want quick access without interrupting your workflow

# ⚡ FR Keyboard Shortcuts 

## 🎯 Purpose

This document provides essential keyboard shortcuts to improve **speed, efficiency, and workflow** on macOS using the **French (AZERTY) keyboard layout**.

Shortcuts are based on **key position**, but are also shown using **visible FR keys** to avoid confusion.

---

## 🧭 System Navigation

| Action              | Shortcut  | FR Key Reference |
| ------------------- | --------- | ---------------- |
| Spotlight Search    | ⌘ + Space | ⌘ + Espace       |
| Switch Applications | ⌘ + Tab   | ⌘ + Tab          |
| Mission Control     | Ctrl + ↑  | Ctrl + ↑         |
| Show Desktop        | F11       | F11              |

## 🧠 Key Insight

Shortcuts are triggered by:

👉 **physical key position, not the visible character**

This means:

* ⌘ + A still means Select All
* even if you see **⌘ + Q on FR**

---

## ⚡ Practical Usage

* Trust muscle memory (key position)
* Do not rely on visible letters
* Expect visual differences in FR

---

## ⚠️ Common Mistakes

* Thinking shortcuts changed → they did not
* Looking at letters instead of position
* Confusing AZERTY vs QWERTY mapping

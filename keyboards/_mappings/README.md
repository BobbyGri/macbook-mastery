# 🔄 Cross-Layout Mappings

## 🎯 Purpose

This section provides **1:1 key mappings between different keyboard layouts**.

It is designed to help users:

* Type in one layout while thinking in another
* Transition between keyboard layouts
* Build muscle memory across layouts

---

## 🧠 What is a Mapping?

A mapping defines how keys correspond between two layouts.

Example:

* You **see or think in US layout**
* You want to **type in Russian layout**

A mapping shows:

👉 which key you press on your known layout
👉 what character it produces in the target layout

---

## 🧩 Scope

Mappings in this section are intentionally limited to:

* **Base key positions (1:1 mapping)**
* Standard characters (no modifiers)

They do **not** include:

* Option (⌥) combinations
* Shift + Option (⇧ + ⌥) characters
* Advanced or extended symbols

---

## ⚠️ Why this limitation?

To keep mappings:

* Simple
* Learnable
* Focused on muscle memory

For advanced characters, refer to the destination layout documentation.

---

## 🗺️ How to Use

1. Choose a mapping file (e.g., [US→RU](US-to-RU.md))
2. Find the key you are familiar with
3. Identify the corresponding output in the target layout
4. Practice typing using the mapping

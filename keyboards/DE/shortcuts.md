# ⚡ DE Keyboard Shortcuts 

## 🎯 Purpose

This document provides essential keyboard shortcuts to improve **speed, efficiency, and workflow** on macOS using the **German (QWERTZ) keyboard layout**.

Shortcuts are based on **key position**, but are also shown using **visible DE keys** to avoid confusion.

---

## 🧭 System Navigation

| Action              | Shortcut  | DE Key Reference |
| ------------------- | --------- | ---------------- |
| Spotlight Search    | ⌘ + Space | ⌘ + Leertaste    |
| Switch Applications | ⌘ + Tab   | ⌘ + Tab          |
| Mission Control     | Ctrl + ↑  | Ctrl + ↑         |
| Show Desktop        | F11       | F11              |

## 🧠 Key Insight

Shortcuts are triggered by:

👉 **physical key position, not the visible character**

---

## ⚡ Practical Usage

* Most shortcuts feel identical to US
* Differences appear mainly with symbols ([ ], @, etc.)
* Y/Z swap does **not affect shortcuts significantly**

---

## ⚠️ Common Mistakes

* Confusing Y and Z positions while typing
* Expecting direct access to [ ] { }
* Forgetting Option-based combinations for symbols

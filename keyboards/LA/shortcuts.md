# ⚡ LA Keyboard Shortcuts 

## 🎯 Purpose

This document provides essential keyboard shortcuts to improve **speed, efficiency, and workflow** on macOS using the **Latin American Spanish keyboard layout (LA)**.

Shortcuts are based on **key position**, but are also shown using **visible LA keys** to avoid confusion.

---

## 🧭 System Navigation

| Action              | Shortcut  | LA Key Reference |
| ------------------- | --------- | ---------------- |
| Spotlight Search    | ⌘ + Space | ⌘ + Espacio      |
| Switch Applications | ⌘ + Tab   | ⌘ + Tab          |
| Mission Control     | Ctrl + ↑  | Ctrl + ↑         |

## 🧠 Key Insight

Shortcuts are triggered by:

👉 **physical key position, not the visible character**

---

## ⚡ Practical Usage

* Most shortcuts behave exactly like US
* Differences are minimal compared to FR or DE
* Symbol keys (like @, { }) may require Option

---

## ⚠️ Common Mistakes

* Expecting US symbol positions
* Forgetting Option for characters like @
* Thinking shortcuts change with layout
* Overusing the mouse instead of shortcuts
